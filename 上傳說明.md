### 1. 檢查目前狀態
```bash

```

### 2. 檢查要上傳的檔案
```bash

git add --dry-run .

# 確認 .gitignore 生效
git check-ignore configs/ tests/ backups/ .env
```

### 3. 暫存要上傳的檔案
```bash

```
